Context

Atlas is a small REST API that aggregates hiking trail data from open datasets and serves it to a mobile client. Stack: FastAPI + PostgreSQL, deployed on a self-hosted VPS.

Triptych

Problem

Trail search endpoint timed out on queries covering more than one region.

Reasoning

The query joined three tables without an index on region_id; every search scanned the full trails table.

Solutions

Added a composite index, capped results at 200 per page. p95 latency back under 300ms.
